# 연산자 공부
## 종류
### 산술 연산자 (이항)
- \+ : 덧셈
- \- : 뺄셈
- \* : 곱셈
- / : 나눗셈
- % : 나머지

### 부호 연산자 (단항)
- \+ : 양수
- \- : 음수

### 문자열 연산자 (이항)
- \+ : 문자열 연결

### 대입 연산자 (이항)
- = : 대입
- \+= : 덧셈 후 대입
- \-= : 뺄셈 후 대입
- \*= : 곱셈 후 대입
- /= : 나눗셈 후 대입
- %= : 나머지 후 대입

### 증감 연산자 (단항)
- ++ : 증가
- -- : 감소

### 비교 연산자 (이항)
- == : 같다
- != : 같지 않다
- a > b : a가 b보다 크다
- a < b : a가 b보다 작다
- a >= b : a가 b보다 크거나 같다
- a <= b : a가 b보다 작거나 같다
- __instanceof : 객체가 특정 클래스의 인스턴스인지 확인__

### 논리 연산자 (이항4, 단항2)
- && : 논리 AND
- || : 논리 OR
- & : 비트 AND
- | : 비트 OR
- ! : 논리 NOT
- ~ : 비트 NOT

### 조건 연산자 (삼항)
- (조건식) ? a : b
- 조건식이 참이면 a, 거짓이면 b
